# Local Setup Instructions

Welcome to your offline POS project!

We have included scripts to make starting this application on your local computer as easy as double-clicking a file.

## Prerequisites

You must have **Node.js** installed on your computer. 
If you don't have it, download and install it for free from [https://nodejs.org/](https://nodejs.org/) (we recommend the LTS version).

## How to Start the App

### Windows Users
1. Download and extract the project ZIP folder.
2. Inside the folder, double-click on `start-windows.bat`.
3. Wait for it to finish installing (only required the first time) and boot the server.
4. Your browser should open, or you can manually go to `http://localhost:8080`.

### Mac and Linux Users
1. Download and extract the project ZIP folder.
2. Open your terminal app, and go to the folder where you extracted the project.
3. Make the script executable by running: `chmod +x start-mac-linux.sh`
4. Run the script: `./start-mac-linux.sh`
5. Open your web browser and go to `http://localhost:8080`.

---
*Note: We have changed the local port to **8080** to avoid conflicts with other applications running on your machine.*
